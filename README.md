# Vue 3.5 + Vite + Element Plus + Tailwind CSS

一个现代化的 Vue 3.5 项目模板，集成了 Vite、Element Plus 和 Tailwind CSS。

## 技术栈

- **Vue 3.5** - 渐进式 JavaScript 框架
- **Vite 5** - 下一代前端构建工具
- **Element Plus** - 基于 Vue 3 的组件库
- **Tailwind CSS** - 实用优先的 CSS 框架

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

```
table-com/
├── src/
│   ├── App.vue          # 根组件
│   ├── main.js          # 入口文件
│   └── style.css        # 全局样式
├── index.html           # HTML 模板
├── vite.config.js       # Vite 配置
├── tailwind.config.js   # Tailwind 配置
├── postcss.config.js    # PostCSS 配置
└── package.json         # 项目配置
```

## 特性

- ✅ Vue 3.5 Composition API
- ✅ Vite 5 快速构建
- ✅ Element Plus 完整集成
- ✅ Tailwind CSS 样式系统
- ✅ 图标自动注册
- ✅ 路径别名配置 (@/src)
- ✅ 开发服务器自动打开

## 开发规范

- 所有 `.vue` 文件使用 `<script>` 标签（JavaScript 语法）
- 使用 Composition API 进行组件开发
- 遵循 Element Plus 组件使用规范
- 使用 Tailwind CSS 进行样式开发
